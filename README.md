# Разнесение Росстата на сетку 1×1 км

Задача: взять региональный показатель Росстата (один на весь субъект РФ)
и разнести его по сетке 1×1 км на территории региона. Делить просто по
площади нельзя - это никак не отражает реальную неравномерность. Мы
опираемся на внешние пространственные признаки (население, расстояния
до городов, инфраструктура и т.п.)

## Команда

- Фёдор Кравченко - метод затухания

- Екатерина Неяскина - модель максимальной энтропии

## Пилот

Три региона разного типа:

- Московская область - агломерация,
- Краснодарский край - агропромышленный регион,
- центральная Якутия (бокс 400×400 км вокруг Якутска) - редкозаселённый
  регион. Для всей Якутии 1×1 км не помещается в память, для неё
  отдельно построена сетка 10×10 км

Показатели Росстата:

- Y477090007 - объём отгруженных товаров,
- Y477120029 - продукция сельского хозяйства.

## Методы

1. **Baseline** - равномерное распределение по площади ячеек
2. **v1 (затухание)** - вес ячейки убывает с расстоянием до ближайшего
   города
3. **v2 (WorldPop)** - вес = население в ячейке (растр WorldPop 100 м)
4. **v3 (регрессия + WorldPop)** - регрессия `log(отгрузка) ~ log(население) +
   log(гор.население) + log(инвестиции)` на 85 регионах. Коэффициент при
   городском населении (2.77) применяется как эластичность:
   `вес ячейки = население^2.77`
5. **v4 (максимальная энтропия)** — отдельный ноутбук `04_maxent.ipynb`.
   Распределение с максимальной энтропией при ограничениях на средние признаки
   ячеек. Код: `src/spatial_maxent.py`, подготовка входов: `src/maxent_io.py`

## Что получилось (метрики Джини и top-10% по ячейкам)

| Метод | МО | Краснодар | Якутия (центр) |
|---|---|---|---|
| Baseline | 0.018 / 10.2% | 0.014 / 10.1% | 0.005 / 10.1% |
| v1 (затухание) | 0.349 / 29.5% | 0.303 / 26.9% | 0.331 / 29.7% |
| v2 (WorldPop) | 0.91 / 62.1% | 0.969 / 56.1% | 0.995 / 61.2% |
| v3 (регрессия×WP) | 0.993 / 99.1% | 0.996 / 96.4% | 0.999 / 96.6% |
| v4 (MaxEnt, строгий) | 0.993 / 100.0% | 0.997 / 100.0% | 0.999 / 100.0% |
| v4 (MaxEnt, сглаженный) | 0.924 / 89.6% | 0.971 / 99.2% | 0.977 / 98.1% |

Регрессия: R² = 0.84 in-sample, 0.78 ± 0.05 на 5-fold CV. Главный
коэффициент - городское население

## Структура

```
.
├── README.md
├── requirements.txt
├── src/
│   ├── spatial_maxent.py # v4 — модель максимальной энтропии
│   └── maxent_io.py      # загрузка сеток и признаков для v4
├── notebooks/
│   ├── 01_data.ipynb     # разведка Росстата, выбор показателей и пилотов
│   ├── 02_grid.ipynb     # сетка, baseline–v3, метрики, карты
│   ├── 03_worldpop.ipynb # подготовка растра WorldPop
│   └── 04_maxent.ipynb   # v4 — максимальная энтропия
├── data/
│   ├── raw/
│   └── processed/        # сетки в gpkg, обрезанные растры
└── results/              # PNG-карты и CSV со сводными метриками
```

### Запуск v4 (максимальная энтропия)

1. Выполнить `02_grid.ipynb` (сетки `grid_*_1km.gpkg`) и `03_worldpop.ipynb` (растры `pop_*_z.tif`).
2. Открыть и выполнить `04_maxent.ipynb` целиком.
3. Результаты: `results/maxent_metrics.csv`, карты `results/maxent_*.png`, сетки `results/maxent_grids/`.

Повторный запуск быстрее: признаки кэшируются в `data/processed/grid_*_features.gpkg`.
